# Smart Meter Automation Demo Project

This project demonstrates a sample automation framework using Robot Framework.

## Technologies Used

- Robot Framework
- SeleniumLibrary
- RequestsLibrary
- Python

## Project Structure

smart-meter-automation
- data
- resources
- tests
- results

## Test Types

API Tests  
UI Tests  
Integration Tests  
Smoke Tests  

## Running the Tests

Run all tests with:

robot -d results tests/

## Reports

After execution, reports are generated in the results folder:

report.html  
log.html  

## Purpose

This project demonstrates automation skills including:

API testing  
UI automation  
data-driven testing  
test reporting  
framework structure
